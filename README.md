# Classic Games Collection - Pong & Snake

A collection of classic games implemented in Python using the Turtle graphics library. Features two timeless games: Pong and Snake, built as programming practice projects.

### 1. Pong Game
The classic two-player arcade game where players control paddles to bounce a ball back and forth.

**Features:**
- Two-player gameplay
- Score tracking
- Ball physics with collision detection
- Paddle controls for both players
- Classic retro aesthetic

**Controls:**
- Player 1 (Left Paddle): W (up) / S (down)
- Player 2 (Right Paddle): Up Arrow / Down Arrow

### 2. Snake Game
The iconic game where you control a snake to eat food and grow longer without hitting walls or yourself.

**Features:**
- Snake growth mechanics
- Food spawning system
- Collision detection (walls and self)
- Score tracking
- Increasing difficulty as snake grows

**Controls:**
- Arrow Keys: Up, Down, Left, Right
